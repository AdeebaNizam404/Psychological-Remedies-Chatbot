# Psychological Remedies Chatbot
Final Year Project: Psychological Remedies Chatbot based on Sentiment Analysis An intelligent chatbot that analyzes user emotions using NLP techniques and provides personalized psychological support, coping strategies, and mental wellness suggestions.

##  How It Works
1. User enters text input
2. Sentiment Analysis model processes the input
3. Emotion is classified (Positive / Negative / Neutral)
4. Chatbot responds with suitable psychological remedies

 ##  Example Interaction

User: I feel very stressed and anxious  
Bot: I'm here for you Try taking slow deep breaths. Would you like a quick relaxation exercise?

##  Project Impact
- Helps users manage stress and emotions
- Promotes mental well-being awareness
- Demonstrates real-world application of NLP
